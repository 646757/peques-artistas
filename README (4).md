# 🎨 Peques Artistas

Aplicación móvil para niños donde pueden expresar su creatividad dibujando, usando stickers y guardando sus obras de arte.

---

## 📱 Funciones

1. 🔐 Inicio de Sesión
2. 🎨 Lienzo de Dibujo
3. ✏️ Pinceles y Tamaños
4. 🌈 Fondos de Colores
5. ⭐ Stickers
6. 🖼️ Galería
